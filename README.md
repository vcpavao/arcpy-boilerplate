**arcpy-boilerplate**

This repository stores some of my GIS scripts and project samples.

**Contents:**

map-examples - folder containing maps I have created in ArcGIS Pro

massdep_wells_estimates - folder containing some of my work under Professor Nelson da Luz, which was utilized by the Massachusetts DEP Well Drillers Program

hunting-buffers-workflow.ipynb - Jupyter Notebook file for calculating hunting buffers in conservation lands based on proximity to structures and roads. This work was completed for Kestrel Land Trust.

ma-towns-join-and-merge.ipynb -  Jupyter Notebook file for merging parcel geometries with assessors' table information across multiple towns

taxparcels-intersecting-biomap.ipynb - Jupyter Notebook file that preserves tax parcels that intersect with the MassGIS BioMap layer, highlighting ones suitable for conservation efforts
